- B0: Tải và cài đặt Docker

- B1: Giải nén thư mục này, mở Ternimal hoặc PowerShell, cd vào thư mục "clc" này rồi chạy lệnh phía dưới (Lần đầu chạy khá lâu do phải tải file xuống, các lần sau sẽ rất nhanh):
docker-compose up

- B2: Sau khi hoàn tất, chạy lệnh sau:
docker exec -d clc_server_1 git -C /clc_server pull ; docker exec -d clc_server_1 pip3 install -r /clc_server/requirements.txt ; docker exec -d clc_server_1 python3 /clc_server/manage.py runserver 0.0.0.0:8000
